# eosjs-demo
根据eosjs https://github.com/EOSIO/eosjs 衍生的脚本



## 使用说明

 1. 使用 `npm install` 安装依赖

 2. 将./config.json 进行相应配置

 3. 转账` node tansfer.js`

 4. 获取账户信息` node account.js`
